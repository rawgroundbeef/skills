# Product Website

Build, audit, or rewrite a product marketing site so the right visitor can
understand the offer, believe it, act on it, and remember it.

The skill treats positioning, proof, pricing, accessibility, performance, and
conversion as connected product decisions. It inspects the real repository and
available evidence before proposing copy or implementation changes, and it
never invents capabilities, customers, metrics, or competitive claims.

## What it covers

- Audience, job, outcome, differentiation, and primary conversion event.
- A message spine and one dominant promise and action.
- Page structure, demonstration, proof, objections, pricing, and social sharing.
- Prioritized audits across positioning, hierarchy, proof, action, trust, and
  distribution.
- Repository implementation with responsive, accessible, performant behavior.
- Verification of the rendered page, CTA destination, metadata, assets, and
  every factual claim.

## Try it

```text
Use $product-website to audit this landing page and prioritize what is hurting it.
```

```text
Use $product-website to rewrite and implement a clearer homepage for this product.
```
